# Pistols Arena Online
This game is an online version of my previous game "Pistols Arena 3d", I was gonna update it but decided to make a new version with better arena instead. 
<img width="1147" height="637" alt="Screenshot from 2026-06-30 15-27-32" src="https://github.com/user-attachments/assets/590a5cc6-b758-4188-bf43-8d77b3031bdb" />
the game starts with a main menu where you press play and host a server, if you'd enabled upnp on your router, and pressed host, a new server will be created and will send an ip address in a terminal window. You can share the game with a friend, send him the ip address and he can easily join.
<img width="830" height="369" alt="image" src="https://github.com/user-attachments/assets/5eabe4be-d1ab-44d0-9142-3aa57ed50cbe" />

## Controls
WASD to move around, left click to shoot.
## How to play
From your device, you click play then host and the server gets created and spawns you in the middle of the arena.
to make your friends be able to join, you send them the ip address appeared in your terminal window, thye click play, enter the ip addres then join, and you're ready to go.
The arena is specially designed for pvp with walls and different floors so that the game becomes more funny to play.

### Further Information
This game was made using godot engine, all the charaacters and instances are manually made using CSG (constructoive solid geometry) in godot. (except for the shoot effect and cross hair)
